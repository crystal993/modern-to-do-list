## My Todo List 

원티드 프리온보딩 코스 지원을 위한 사전과제 입니다. 

## 📅 프로젝트 기간   

기간 : 2022년 10월 11일 ~ 2022년 10월 12일

<br>
<hr>
<br>

### ✨ 배포 링크 
https://wanted-pre-onboarding-frontend-jade.vercel.app/

### ✨ 설치 및 실행
```
$ yarn install
$ yarn start
```
지금 프로젝트를 클론한 뒤 위 순서대로 실행하면 localhost:3000 포트에서 프로젝트가 실행됩니다.
